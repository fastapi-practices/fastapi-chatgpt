# FastAPI ChatGPT

基于 open-interpreter 构建 FastAPI ChatGPT 服务

## First steps

1. Install dependencies

    ```bash
    pip install -r requirements.txt
    ```

2. Set OpenAI API key

   Mac/Linux:

    ```text
    export OPENAI_API_KEY=your_api_key
    ```

   Windows:

    ```text
    setx OPENAI_API_KEY your_api_key
    ```
3. Run the server
