# FastAPI ChatGPT

Build the FastAPI ChatGPT API on top of open-interpreter

## First steps

1. Install dependencies

    ```bash
    pip install -r requirements.txt
    ```

2. Set OpenAI API key

   Mac/Linux:

    ```text
    export OPENAI_API_KEY=your_api_key
    ```

   Windows:

    ```text
    setx OPENAI_API_KEY your_api_key
    ```
3. Run the server
